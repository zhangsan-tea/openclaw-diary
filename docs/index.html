1. 什么是“龙虾”？

常规 AI 是入口，龙虾（OpenClaw）是数字员工。

💻 电脑与网络权限

能真实接触电脑环境，读文件、调接口、控制浏览器，不再局限于聊天框。

🧠 持久记忆系统

拥有长期记忆与上下文追踪，能逐步理解你、积累经验、延续任务进度。

🔄 持续自动执行

支持定时任务、自动巡检、主动提醒，24小时待命，无需每次重新下指令。

📱 远程全栖接管

通过 Telegram / 企微 等聊天软件远程操控，多身份 Agent 协同工作。

6. 省钱秘籍：Token 优化战

零散提问极耗 Token。稍复杂的任务，模型高一寸，工程胜一丈。

  • 理清再问（分步法）：先用便宜对话框把需求理顺，再打包扔给龙虾干活。
  • 拦截长文本：绝对禁止全量读取几万字的日志，强制使用摘要或头尾截取。
  • 工作区大扫除：根目录只留核心文件，历史草稿必须移入 archive/，否则每次起步多耗几万 Token。
  • 三步切断法：长会话启用 /new 前，先让系统生成 300 字精简摘要存档，/new 之后重载摘要，无缝续接并省下 90% 费用。

7. 安全防线

🛡️ 物理隔离密钥

填写第三方 API 密钥时，去强隔离、无记忆的独立环境操作，避免残留在日常 MEMORY.md 中。

🔍 ClawScan 体检

定期运行安全扫描助手，排查漏洞。风险评估先行，低风险快跑，高风险暂缓。

📦 手搓自研 Skill

避免安装不明 MCP。核心能力（如 ppt-to-text, token-optimizer）尽量由本地进化官手搓生成，可控度最高。

8. 核心方法论

使用 AI 需要有“往后退一步”的意识。放弃完美提问，慢就是快。
  • 方向不清时退一步：写文章没思路？先让 AI 出框架建议。
  • 不知提问时退一步：倒回去问“我该怎么提问？”，或讲大概情况问它“你还需要我提供什么？”
  • 责任不外包，执行敢放权：“我授权你，你自己想办法处理这个问题。”
  • 不要用固有想法套住 AI：让自己发现的规律晚点给，先让 AI 自己分析数据找规律，有新发现再叠加。
